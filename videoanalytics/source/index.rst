Documentación de videoanalytics 
===============================

*videoanalytics* es una librería para prototipado de solución de videoanalítico.

- Define interfaces para generación y consumo de datos.
- Grafos / contexto compartido.
- Una biblioteca de bloques reutilizables:
	- Fuentes:
		- Lectura de video de archivo.
		- Lectura de video de cámara.
	- Sumideros:
		- Detección de objetos
		- Anotación de videos
		- Seguimiento de objetos
		- Registro en archivos y bases de datos.
